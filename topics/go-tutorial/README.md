# go-tutorial

## 結論（TL;DR）

- Goの基本文法と標準的な書き方を章ごとに整理する
- 各章は `code/chXX-*/` に README と `main.go` を置く

## 目的 / 背景

- Goのチュートリアル内容を、最小サンプルとメモで素早く参照できるようにする

## メモ

- 詳細な調査ログは `notes.md` に残す
- 章別のメモとコードは `code/chXX-*/` に整理する

## コード例

- `code/` に章別の最小サンプルを配置
- 実行方法:
  - `cd topics/go-tutorial/code/ch01-getting-started`
  - `go run main.go`
